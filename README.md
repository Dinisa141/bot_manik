# Бот для записи на маникюр

Telegram-бот для маникюрного салона с возможностями:

- приветствие и предложение подписаться на канал;
- сбор имени и телефона через кнопку «Отправить номер телефона»;
- меню: «Записаться на услугу», «Мои записи», «Отзывы», «Наш телеграмм канал»;
- выбор услуг, даты, времени и мастера;
- просмотр своих записей (и возможность их отменить);
- уведомления менеджеру о новых записях и отзывах в Telegram.

## Где хранятся данные

- **Записи** сохраняются в файле `bookings.json` в папке проекта. Этого достаточно для одного салона и одного бота. Файл можно открыть вручную или перенести на другой компьютер.
- При необходимости можно перейти на **SQLite** (один файл `.db`, удобно для поиска и отчётов) или облачную базу — структура данных в коде к этому готова.


## Установка

1. Установите Python 3.10+.
2. В терминале перейдите в папку проекта:

```bash
cd "c:\Users\Dinis\Desktop\Боты в тг\портфолио ботов\бот для записи на маникюр"
```

3. Установите зависимости:

```bash
pip install -r requirements.txt
```


**Windows (PowerShell):**
```powershell
$env:BOT_TOKEN="ваш_токен_от_BotFather"
python bot.py
```

**Windows (cmd):**
```cmd
set BOT_TOKEN=ваш_токен_от_BotFather
python bot.py
```

**Linux / macOS:**
```bash
export BOT_TOKEN="ваш_токен_от_BotFather"
python bot.py
```

## Запуск

```bash
python bot.py
```

После запуска бот начнёт опрашивать Telegram и отвечать пользователям.
